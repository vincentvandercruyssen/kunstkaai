# Nexwear Future Proof

## Werking
Deze webapp is een kleine minigame rond Nexwear, een recyclebaar kledingconcept met knoppen die kledingstukken transformeren. De speler krijgt drie duurzame challenges en moet binnen vier zetten de juiste combinatie kiezen: mouwlengte, kledinglengte, patroon en een extra vestlaag.

## Problemen en obstakels
Ik had geen bestaande HTML-structuur, dus ik heb het project vanaf nul opgebouwd met een duidelijke gameflow. Ik heb de interacties getest op responsive layout en transitions voor betere UX.

## Context
Deze game past bij Nexwear omdat het product draait om multifunctionele kleding die je kunt aanpassen zonder nieuw textiel te gebruiken. Het ondersteunt de crowdfundingboodschap dat mode duurzaam en slim kan zijn.

## Verantwoording (AI & Bronnen)
Ik heb de interface en code zelf ontworpen met behulp van de VS Code-omgeving. De game gebruikt HTML, CSS en JavaScript, en het concept is gemaakt voor de Weboriëntatie-opdracht "Future Proof".
