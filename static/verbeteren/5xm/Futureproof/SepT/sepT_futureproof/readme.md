Mijn game gaat over het opruimen van plastic flesjes in het bos. je probeert de hoogste score te behalen door de flesjes in de recycle bin te doen. Je krijgt ook munten waarmee je upgrades kan kopen bv dat je sneller word je kan ook werelden kopen voor zo een meer game gevoel te creëren, en je kan ook kiezen uit verschillende game modes.

Ik ben niet veel problemen tegengekomen wel dat mijn game soms niet werkte maar dan heb ik aan de build in ai gevraagt wat er fout is aan de code en dat lost het op en als het er niet uit ziet zoals ik wil dan schrijf ik gewoon een betere prompt aan ai(gemini)

Het past er wel bij voor een deel. Ik heb een flesje bedacht dat vergaat in de natuur. Deze game laat eiglijk zien dat je normale plastic flesjes moet opruimen omdat die niet vergaan en met mijn plastic flesje zou je niet hoeven op te ruimen.

Ik heb chatgpt gebruikt, maar die vond ik niet zo goed dus ben ik naar gemini gegaan daar heb ik veel aanpassingen met gedaan. als er iets fout gingn heb ik gewoon met de build in ai het opgelost, en ik heb alle prompts zelf geschreven.
