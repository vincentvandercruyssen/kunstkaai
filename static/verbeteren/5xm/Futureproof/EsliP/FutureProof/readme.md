# FutureProof

Kleine statische webapplicatie voor demonstratiedoeleinden.

## Beschrijving

Deze repository bevat een eenvoudige frontend met de volgende bestanden:

- [index.html](index.html#L1) — hoofdpagina
- [app.js](app.js#L1) — JavaScript-logica
- [style.css](style.css#L1) — styling

De app is bedoeld als startpunt voor experimenten en onderwijs.

## Vereisten

- Moderne webbrowser
- (optioneel) een lokale static file server voor development

## Installatie & starten

Opties om de app lokaal te draaien:

- Open `index.html` direct in je browser (simpelste manier).
- Met Python (snelle lokale server):

```bash
python -m http.server 8000
# Open http://localhost:8000 in je browser
```

- Met npx (serve):

```bash
npx serve .
# Open de door serve aangegeven URL
```

- Of gebruik de VS Code-extensie **Live Server** en klik op "Go Live".

## Structuur

- `index.html` — HTML-structuur en koppelingen naar CSS/JS
- `style.css` — globale stijlen
- `app.js` — client-side script
