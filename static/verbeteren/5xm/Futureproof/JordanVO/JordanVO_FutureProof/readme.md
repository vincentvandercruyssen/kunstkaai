AquaAura is een slimme douchecoach die gebruikers helpt om water te besparen. De app houdt de douchetijd bij met een timer en geeft via kleuren aan of de gebruiker binnen de aanbevolen tijd blijft. Daarnaast bevat de app een waterbesparingsmodus en een weekoverzicht met statistieken. Tijdens de ontwikkeling waren de grootste uitdagingen het laten werken van de timer, de kleurveranderingen, het waarschuwingsgeluid en de grafieken. Dit project maakt deel uit van een groter concept waarbij een slimme watermeter gebruikers helpt duurzamer om te gaan met water. Voor het ontwikkelen van de code werd gebruikgemaakt van ChatGPT, MDN Web Docs, Google Fonts en online documentatie over HTML, CSS en JavaScript.
