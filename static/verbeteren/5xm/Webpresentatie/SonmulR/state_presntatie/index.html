Webpresentatie

Wat is state?

State is de huidige toestand van een website of programma. JavaScript onthoudt deze informatie.

Voorbeelden van state

  • Aantal keer geklikt
  • Knop aan of uit
  • Tekst die verandert

Waarom is state belangrijk?

Zonder state zou een website niets kunnen onthouden. Elke actie zou opnieuw beginnen.
