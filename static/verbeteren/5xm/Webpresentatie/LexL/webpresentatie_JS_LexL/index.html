Functies herschrijven in JavaScript

Leer hoe je functies slimmer, korter en herbruikbaar maakt

Wat is een functie?

Een functie in JavaScript is een blok code dat je kunt hergebruiken. In plaats van dezelfde code meerdere keren te schrijven, roep je de functie aan.

Standaard functie

Zo ziet een klassieke functie eruit:

function telOp(a, b) {
return a + b;
}


console.log(telOp(2, 3));

Functie herschrijven (arrow function)

Arrow functions zijn korter en moderner. Ze worden veel gebruikt in moderne JavaScript. 

const telOp = (a, b) => {
return a + b;
};

Extra verkorte versie

Als je functie maar één regel heeft, kun je return weglaten. 

const telOp = (a, b) => a + b;
