for-loops (for loops and forEach)

In JavaScript, there are different types of loops, and two of the most popular ones for iterating over data are for and forEach.

for — classic loop

This is a standard loop that you fully control yourself.

How it works:

    let i = 0 — initial value
    i < 5 — condition (as long as it is true, the loop runs)
    i++ — step (increase i by 1)

Pros

    You can use break and continue
    Works with any data structure
    Full control over the index

forEach — array method

forEach is an array method that executes a function for each element of the array.

Syntax

Example:

What is passed:

    element — the current array element
    index — the element’s index
    array — the array itself

Important

    forEach cannot be stopped (break / continue do not work)
    Works only with arrays

Difference between for and forEach

for forEach
You can use break and continue ❌ You cannot stop the loop directly
Works with arrays, objects, strings, NodeLists, and other iterables Only works with arrays (or array-like objects viaArray.from())
More flexible, suitable for complex logic More compact and readable for simple operations
Old syntax, supported in all browsers Modern style, ES5+

When should you use each one?

Use for if you need to:

    stop the loop early
    mplement complex logic

Use forEach if

    simply iterate over an array
    write clean and readable code
