`forEach` — метод массивов

`forEach` — это **метод массива**, который выполняет функцию для каждого элемента массива.

Важно

    `forEach` **нельзя остановить** (`break` / `continue` не работают)
    Работает **только с массивами**

Разница между `for` и `forEach`

for forEach
Можно использовать break и continue ❌ Нельзя прерывать цикл напрямую
Работает с массивами, объектами, строками, NodeList и другими итерируемыми Только с массивами (или массивоподобными объектами через Array.from())
Более гибкий, подходит для сложной логики Более компактный и читабельный для простых операций
Старый синтаксис, поддержка во всех браузерах Современный стиль, ES5+

Когда что использовать?

Используй **`for`**, если нужно:

    рервать цикл
    сложная логика

Используй **`forEach`**, если:

    просто перебрать массив
    написать чистый и понятный код
