DOM selecteren met querySelector

Wat is querySelector?

querySelector is een JavaScript-methode waarmee je één element uit de HTML-pagina kan selecteren. Je gebruikt dezelfde selectors als in CSS.

Hoe werkt het?

Met querySelector selecteer je één element. JavaScript zoekt dan het eerste element dat overeenkomt met die selector. Daarna kan je dat element aanpassen, bijvoorbeeld de tekst, stijl of inhoud. Met querySelectorAll kan je meerdere elementen tegelijk selecteren, zoals alle links of knoppen op een pagina.
