Tekst aanpassen met textContent

Wat is textContent?

textContent is een eigenschap van elk HTML-element. Hiermee kan JavaScript de tekst binnen dat element ophalen of veranderen.

Het gaat enkel om de tekst, niet om de opmaak of HTML.

Waarom bestaat textContent?

Moderne websites moeten interactief reageren op gebruikers. Denk aan:

  • Klikken op knoppen
  • Formuliervelden invullen
  • Status tonen, zoals "Gelukt!"

Dankzij textContent kan een website dit zonder herladen.

Hoe werkt het?

  1. Selecteer het element met querySelector
  2. Gebruik textContent om de tekst te veranderen

// Selecteer een element
const titel = document.querySelector("h2");

// Verander de tekst
titel.textContent = "Nieuwe tekst!";

Wanneer gebruik je textContent?

Voorbeelden uit echte websites:

  • Formulierstatus: “Dankjewel voor je inzending!”
  • Score tonen in een spel
  • Chatberichten toevoegen
  • Tekst aanpassen bij knoppen en menu’s

Overal waar tekst verandert, komt textContent van pas.

Samenvatting

  • textContent verandert de tekst in HTML
  • Handig voor dynamische, interactieve pagina’s
  • Superveel gebruikt in websites, games en apps
