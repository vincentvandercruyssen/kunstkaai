Style/CSS aanpassen in JavaScript

Weboriëntatie – JavaScript concept

Wat is het?

Met JavaScript kan je de opmaak (CSS) van een webpagina aanpassen terwijl de pagina al geladen is. Zo kan je kleuren, groottes of andere stijlen veranderen op basis van gebruikersacties.

Waarom gebruiken we dit?

CSS aanpassen via JavaScript wordt vaak gebruikt voor interactieve websites. Denk aan knoppen, thema’s (donker/licht), waarschuwingen of animaties.

Codevoorbeeld

In JavaScript selecteren we een element en passen we de CSS aan via .style. Het verschil tussen CSS en JavaScript is dat we in JavaScript dingen anders aanspreken en de stijl direct aanpassen via de DOM. Kort gezegd: JavaScript bepaalt wanneer iets verandert, en CSS bepaalt hoe het eruitziet. 


let styleToggle = false;
const box = document.querySelector("#box");
const styleBtn = document.querySelector("#styleBtn");

styleBtn.addEventListener("click", boxToggleStyle);

function boxToggleStyle() {
  styleToggle = !styleToggle;
  if (styleToggle) {
    box.style.backgroundColor = "lightblue";
    box.style.color = "white";
    box.style.borderRadius = "10px";
    box.style.fontSize = "1.2rem";
  } else {
    box.style.backgroundColor = "rgb(170, 187, 193)";
    box.style.color = "black";
    box.style.borderRadius = "0";
    box.style.fontSize = "1rem";
  }
}
