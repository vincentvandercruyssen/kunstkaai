Wat is een event?

Een event is een gebeurtenis die plaatsvindt in de browser. Dit kan gebeuren door een gebruiker of door de pagina zelf.

Voorbeelden van events zijn:

  • klikken op een knop
  • een toets indrukken
  • de pagina laden
  • met de muis over een element gaan

Wat is een event listener?

Een event listener is JavaScript-code die luistert naar een event en iets uitvoert wanneer dat event gebeurt.

Zonder event listeners zou een website niet weten wanneer hij moet reageren op de gebruiker.

Codevoorbeeld

In dit voorbeeld zit er een knop op de pagina. Wanneer je op de knop klikt, verandert de tekst.

Er is nog niet geklikt.

const knop = document.getElementById("klikKnop"); const tekst = document.getElementById("resultaat"); knop.addEventListener("click", function () { tekst.textContent = "Je hebt op de knop geklikt!"; });

Wat gebeurt hier?

JavaScript luistert naar het click-event van de knop. Zodra de gebruiker klikt, wordt de tekst aangepast.
