Events en EventListeners

Hoe JavaScript reageert op acties van de gebruiker

Wat is een event?

Een event is een gebeurtenis die plaatsvindt op een webpagina. Dit kan een actie van de gebruiker zijn, zoals klikken, typen of bewegen van de muis. Maar het kan ook automatisch gebeuren, bijvoorbeeld wanneer de pagina klaar is met laden.

Events zorgen ervoor dat JavaScript weet wanneer er iets gebeurt. Zonder events zou een webpagina statisch zijn en niet reageren op de gebruiker.

Voorbeelden van events:

  • Klikken op een knop
  • Bewegen van de muis
  • Indrukken van een toets
  • Pagina is geladen

Wat is een eventlistener?

Een eventlistener is een stukje JavaScript dat “luistert” naar een event. Zodra dat event plaatsvindt, voert de eventlistener een functie uit.

Je kan het zien als: “Als dit gebeurt, doe dan dat.” Dit maakt een website interactief, omdat er dan direct iets kan veranderen op de pagina.

Eventlisteners worden vaak gebruikt voor:

  • Knoppen
  • Formulieren
  • Animaties
  • Spellen

Waarom zijn events belangrijk?

Events en eventlisteners maken websites interactief. Zonder eventlisteners kan JavaScript niet reageren op de gebruiker. Hierdoor zou een website alleen tekst en afbeeldingen tonen, zonder dat er iets verandert.

Met events kan je dingen doen zoals: inhoud veranderen, pagina-elementen verbergen, nieuwe elementen toevoegen, of zelfs games maken.

Voorbeeld

Deze tekst zal veranderen.

Muiscoördinaten (event + eventlistener)

Beweeg je muis over de pagina en je ziet de exacte x/y-coördinaten.

X: 0 | Y: 0

Codevoorbeeld

/* SLIDE EVENTS */
volgendeBtn.addEventListener("click", () => {
  if (huidigeSlide < slides.length - 1) {
    huidigeSlide++;
    toonSlide(huidigeSlide);
  }
});

vorigeBtn.addEventListener("click", () => {
  if (huidigeSlide > 0) {
    huidigeSlide--;
    toonSlide(huidigeSlide);
  }
});

/* KNOP EVENTS */
klikKnop.addEventListener("click", () => tekst.textContent = "Je hebt op de knop geklikt!");
resetKnop.addEventListener("click", () => tekst.textContent = "Deze tekst zal veranderen.");

/* MUIS EVENTS */
document.addEventListener("mousemove", (e) => {
  coords.textContent = `X: ${e.clientX} | Y: ${e.clientY}`;
});
