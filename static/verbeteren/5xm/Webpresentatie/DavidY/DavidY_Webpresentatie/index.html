Wat is een functie?

Een functie in JavaScript is een stukje code dat je een naam geeft en dat je later kan oproepen om het opnieuw uit te voeren. Je kan het zien als een soort recept of machine die een taak uitvoert.

Het voordeel is dat je code niet telkens opnieuw moet schrijven: je definieert de functie één keer en je kan ze daarna op verschillende plaatsen gebruiken.

Waarom bestaan functies?

Functies zorgen ervoor dat je code korter, leesbaarder en herbruikbaar wordt. In plaats van dezelfde code vijf keer te kopiëren, schrijf je één functie en roep je die vijf keer op.

Dat maakt fouten zoeken makkelijker en je kan één functie aanpassen in plaats van overal losse stukjes code te moeten wijzigen.

Hoe schrijf je een eenvoudige functie?

Een klassieke functie ziet er zo uit:

                
// definitie van de functie
function begroet() {
console.log("Hallo, welkom op mijn pagina!");
}

// de functie oproepen (uitvoeren)
begroet();

function is het sleutelwoord, begroet is de naam van de functie, en tussen de { } staat de code die wordt uitgevoerd wanneer je de functie oproept.

Functies met parameters en return

Functies kunnen ook parameters (input) krijgen en een return-waarde (output) teruggeven.

  
                
function telOp(getal1, getal2) {
const som = getal1 + getal2;
return som;
}

const resultaat = telOp(3, 5); // resultaat is 8
console.log(resultaat);

getal1 en getal2 zijn parameters. De functie telt ze op en return stuurt de uitkomst terug, zodat je die kan bewaren in een variabele of tonen op het scherm.

Voorbeeld van een functie

Hieronder zie je een voorbeeld van een interactieve functie. Jij geeft twee getallen in en een functie rekent de som uit en toont het resultaat in de pagina.

De som verschijnt hier.

Code van de voorbeeld (JavaScript)

                
// functie die twee getallen optelt
function berekenSom(a, b) {
  const resultaat = a + b;
  return resultaat;
}

// elementen uit de HTML selecteren
const input1 = document.querySelector("#getal1");
const input2 = document.querySelector("#getal2");
const knop = document.querySelector("#bereken-knop");
const uitkomstTekst = document.querySelector("#uitkomst-tekst");

// eventlistener om de functie uit te voeren wanneer je op de knop klikt
knop.addEventListener("click", function () {
  const waarde1 = Number(input1.value);
  const waarde2 = Number(input2.value);
  const som = berekenSom(waarde1, waarde2);

  uitkomstTekst.textContent = "De som is: " + som;
});
