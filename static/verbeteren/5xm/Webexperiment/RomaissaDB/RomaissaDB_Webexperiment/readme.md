# Cosmic Pulse – Webexperiment

## Wat ik gemaakt heb en waarom
Ik heb een generatief webexperiment gemaakt dat "Cosmic Pulse" heet. Het is een bewegend veld van lichtpuntjes (partikels) die over het scherm zweven, lijnen vormen en reageren op de muis. Het is bedoeld als een hypnotiserende, bijna kosmische visual die op een groot scherm blijft bewegen en bezoekers uitnodigt om ermee te spelen.

## Welke bronnen en prompts/AI ik gebruikte
Ik heb inspiratie gehaald uit het idee van "particle systems" en "generative art" in JavaScript. Voor de concrete uitwerking heb ik een AI-co-piloot gebruikt (LLM) om de basisstructuur van het partikel-systeem, de animatieloop en de interactie (muis, klik-puls) te genereren en te verfijnen. Ik heb zelf de parameters, kleuren, aantallen en gedrag aangepast tot de vibe klopte.

## Wat ik geleerd heb
Ik heb geleerd hoe je met het canvas-element en JavaScript een animatieloop maakt met `requestAnimationFrame`, hoe je een partikelklasse opbouwt, en hoe je beweging, kleur en levensduur kan combineren tot een generatief systeem. Ook heb ik beter begrepen hoe je kleine interacties (muispositie, klikken) kan koppelen aan visuele effecten.

## Wat ik anders zou doen als ik meer tijd had
Met meer tijd zou ik geluid toevoegen via de Web Audio API, zodat de animatie reageert op muziek of omgevingsgeluid. Ook zou ik misschien een instellingenpaneel maken (bijvoorbeeld met sliders) waarmee bezoekers zelf het aantal partikels, de kleuren of de intensiteit van de puls kunnen aanpassen. Daarnaast zou ik willen experimenteren met 3D (Three.js) of shaders voor nog extremere visuals.
