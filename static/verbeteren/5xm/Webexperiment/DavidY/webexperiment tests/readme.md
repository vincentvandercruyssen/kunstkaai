Ik maakte vier verschillende webexperimenten: een werkende 8‑ball, een ruimte met zwevende constellaties die reageren op de cursor, een bloemachtig wezen dat nieuwsgierig is over het muis maar bang kan worden als het te snel beweegt, en een CRT‑monitor die het webcam toont met enkele verschillende filters. 
Ik wal een webexperiment maken rond deze 4 dingen : 
- Audio
- Webcam
- Visueel
- Interactiviteit
Als ik geen idee had voor ene, zou ik het combineren met een andere om toch iets ermee te doen.

Voor alle experimenten heb ik Copilot gebruikt om ze te maken, maar alle ideeën kwamen van mij. 

Ik heb geleerd hoe zelfs het kleinste fout veel meer impact heeft op het functionaliteit van het code dan ik origineel dacht.
 
Met meer tijd zou ik de uiterlijk van alle paginas meer uitwerken, vooral bij de 8-ball en de bloem, zodat hun paginas meer personaliteit heeft. Ook zou ik de CRT‑monitor uitbreiden met meer filters om mee te spelen.
