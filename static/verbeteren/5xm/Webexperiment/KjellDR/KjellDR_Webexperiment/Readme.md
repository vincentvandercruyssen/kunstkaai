Wat heb ik gemaakt en waarom

Ik heb een interactief webexperiment gemaakt dat reageert op geluid, muis en webcam. Het toont een psychedelische achtergrond met explosies en tekst die groter wordt wanneer je lawaai maakt.
Ik wilde iets maken dat meteen de aandacht trekt en mensen uitnodigt om ermee te spelen zonder uitleg.

Welke bronnen en prompts/AI ik gebruikte

Ik heb ChatGPT gebruikt om mijn code te verbeteren en problemen op te lossen. Daarnaast heb ik inspiratie gehaald uit CodePen en de documentatie van de Canvas API en Web Audio API (MDN).
Ik heb verschillende voorbeelden gecombineerd en aangepast tot één werkend geheel.

Wat heb ik geleerd

Ik heb geleerd hoe je microfoon- en webcaminput gebruikt in een website en hoe je die koppelt aan visuele effecten.
Ook heb ik geleerd hoe je animaties maakt met canvas en hoe je verschillende technieken samen laat werken.

Wat zou ik anders doen met meer tijd

Ik zou meer visuele effecten toevoegen, zoals glitch-effecten en 3D met Three.js.
Ook zou ik het geheel optimaliseren zodat het nog vloeiender draait en er nog spectaculairder uitziet.
