Kernideeën

Wat maakt een webpagina
tot een ervaring?

⬜

Leegte als taal

Witruimte is geen afwezigheid. Het is de pauze tussen noten — het geeft betekenis aan wat er wél staat.

↔

Beweging als richting

Elk element op het scherm beweegt van ergens naar ergens. Niets verschijnt zomaar — alles heeft een weg.

◑

Licht & donker

Contrast is niet enkel een kleurkeuze. Het is een verhouding tussen aandacht en rust, tussen voor en achter.

⌨

Code als handschrift

Elke regel code is een keuze. De manier waarop je schrijft vormt uiteindelijk wat de bezoeker ziet en voelt.

○

Vorm volgt gevoel

Ronde hoeken voelen anders dan scherpe. Serif tegenover sans-serif. Elk detail heeft een emotionele lading.

∿

Ritme & herhaling

Een pagina heeft cadans. Herhaalde patronen, consistente spacing — ze bouwen verwachting op en doorbreken die.

Interface

De browser als canvas

Een webpagina is nooit af. Ze ademt mee met de gebruiker, past zich aan, reageert. Het verschil tussen een ontwerp en een ervaring is interactie.
