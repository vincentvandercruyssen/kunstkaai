# VoornaamA_Webexperiment — Visuele Dobbelsteen

Wat ik gemaakt heb en waarom
- Een visueel en autonoom draaiend webexperiment: een grote, glanzende dobbelsteen (canvas) die op het scherm beweegt, af en toe automatisch rolt en bij klikken of spatie handmatig rolt. De bedoeling is om bezoekers te laten stoppen: het is rustgevend maar verrassend wanneer de dobbelsteen rolt.

Welke bronnen en prompts/AI ik gebruikte
- Vibecoding: ik gebruikte een AI-assistent om het project te genereren en samen te stellen (HTML/CSS/JS). Inspiratie uit creatieve-coding voorbeelden (gradients, p5.js-achtige esthetiek). Geen externe libraries toegevoegd.

Wat ik geleerd heb
- Hoe je een autonome animatie combineert met gebruikersinteractie op Canvas. Kleine WebAudio-klikken toevoegen en geschiedenis in de UI bijhouden.

Wat ik anders zou doen met meer tijd
- Beter 3D-effect met WebGL/Three.js, extra stellingen (tekst op de zijvlakken) die met API's of geluid veranderen, en meer toegankelijke bediening (touch gestures, grotere touch targets).
