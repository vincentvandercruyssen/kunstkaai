Mijn Schoolprojecten in Kleur

    Goedemiddag iedereen,
    Vandaag laat ik mijn schoolprojecten zien die ik tijdens mijn opleiding heb gemaakt. Om alles duidelijk en overzichtelijk voor te stellen, heb ik mijn projecten verdeeld in vier verschillende kleuren. Elke kleur hoort bij een bepaald onderdeel van mijn opleiding.
  • Groen staat voor beeldverwerking
  • Blauw staat voor webontwikkeling
  • Rood staat voor typografische vormgeving
  • Geel staat voor vectorieel tekenen
      • Door met kleuren te werken, kunnen jullie gemakkelijker volgen bij welk vak of onderdeel elk project hoort. Op deze manier krijgt elk project ook een eigen plaats binnen mijn presentatie.
      Ik ga jullie nu stap voor stap mijn projecten laten zien en tonen waar ik tijdens mijn schoolwerk aan heb gewerkt.
