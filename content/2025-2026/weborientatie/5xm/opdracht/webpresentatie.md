## Introductie

In deze opdracht maak je **cursusmateriaal** in de vorm van een **webpresentatie** rond één **JavaScript(programmeer)-concept**.  
Dat kan een concept zijn dat we **reeds gezien hebben** in de les, of een **nieuw JavaScript-onderwerp** dat jij graag verder wil uitdiepen.

Het doel is niet om iets spectaculairs te bouwen, maar om **zelf goed te begrijpen** hoe het concept werkt, en het **duidelijk uit te leggen** aan anderen.

Je webpresentatie moet zo opgebouwd zijn dat:
- jij ze in **ongeveer 3 minuten** kan toelichten
- een klasgenoot het concept kan **leren** door jouw pagina te bekijken

## Keuze van onderwerp

Kies **één** onderwerp uit de onderstaande lijst. Elke leerling kiest een **ander** onderwerp, zodat er voldoende variatie is.

Mogelijke onderwerpen zijn onder andere:

- Variabelen en datatypes
- State in JavaScript
- If-else-structuren
- Events en eventlisteners
- DOM selecteren met `querySelector`
- Functies schrijven en hergebruiken
- Tekst aanpassen met `textContent`
- CSS aanpassen via JavaScript
- Arrays
- For-loops

Wil je **een ander JavaScript-concept** voorstellen, dan vraag je dat **eerst** even na.

## Wat maak je?

Je maakt een **eenvoudige webpresentatie** die het gekozen concept uitlegt.

Je webpresentatie bevat:
- een **korte inleiding**: wat is dit concept
- een **duidelijke uitleg in eigen woorden**
- **minstens één codevoorbeeld**
- **visuele ondersteuning** waar nuttig (tekst, schema, voorbeeld, kleur, eenvoudige demo)

De nadruk ligt op **begrip en helderheid**, niet op design of complexiteit.

## Structuur van je project

Werk in je vakmap voor **Weboriëntatie**.

```
VoornaamA_Webpresentatie/
├── index.html
├── style.css
├── app.js
└── images/
```

Gebruik duidelijke bestandsnamen en volg steeds dezelfde structuur.

## HTML

- Gebruik **semantische HTML**:
  - `header`
    - `nav`
  - `main`
    - `section`
  - `footer`
- Gebruik:
  - één `h1`
  - meerdere `h2`
  - eventueel `h3`
  - `p`
  - `img`

## CSS

- Alle opmaak staat in `style.css`
- Zorg voor:
  - leesbare tekst
  - voldoende contrast
  - een rustige, overzichtelijke lay-out

## JavaScript

- Gebruik JavaScript **enkel waar het concept dat vraagt**
- Je code moet:
  - eenvoudig
  - leesbaar
  - correct benoemd zijn

## Mondelinge toelichting

Je licht je webpresentatie kort toe in de klas:

- Duur: **ongeveer 3 minuten**
- Je legt uit:
  - wat het concept is
  - waarom het bestaat
  - hoe jouw voorbeeld werkt
- Je **leest niet voor**, je **legt uit**.

## Bronnen

- Noteer onderaan je pagina:
  - gebruikte websites
  - documentatie
  - tutorials
- Kopiëren zonder bronvermelding is niet toegestaan.

## Puntenverdeling

- **Structuur en netheid** Correcte map, juiste bestanden, duidelijke opbouw
- **HTML** Semantische structuur, logische indeling, correcte koppen
- **JavaScript-concept** Correct uitgelegd, juiste terminologie, eigen woorden
- **Didactische kwaliteit** Begrijpelijk voor klasgenoten, heldere voorbeelden
- **Mondelinge toelichting** Duidelijk, voorbereid, binnen de tijd

## Doel van deze opdracht

Deze opdracht helpt je om:
- JavaScript beter te **begrijpen**
- technische concepten te **verwoorden**
- stap voor stap te leren **uitleggen wat code doet**
